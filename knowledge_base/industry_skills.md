# Industry Skills & Learning Paths

This document maps core skills and upskilling pathways for common roles. Use these to suggest specific certifications and training to candidates.

## 1. Software Engineering / Web Development
* **Core Technical Skills**: Python, JavaScript, Java, C++, TypeScript, React, Node.js, SQL, NoSQL, Git, Docker, Kubernetes, CI/CD pipelines, AWS/GCP/Azure.
* **Core Soft Skills**: Agile methodology, system design, code reviews, collaboration, technical communication, problem-solving.
* **Upskilling Pathways & Resources**:
  - AWS Certified Solutions Architect (for cloud infrastructure gaps)
  - Docker & Kubernetes Complete Guide on Udemy/Coursera (for DevOps gaps)
  - System Design Interview Guide (for senior engineering candidates)
  - Frontend Masters / React Certification (for frontend developers lacking TypeScript/modern React)

## 2. Data Science & Machine Learning
* **Core Technical Skills**: Python, R, SQL, Pandas, NumPy, Scikit-Learn, TensorFlow, PyTorch, Tableau, PowerBI, ETL, A/B Testing, Machine Learning, Deep Learning.
* **Core Soft Skills**: Data storytelling, stakeholder communication, business acumen, research methodology.
* **Upskilling Pathways & Resources**:
  - DeepLearning.AI Machine Learning Specialization (for ML theory and math)
  - Google Data Analytics Certificate (for Excel, SQL, Tableau, R)
  - Databricks Certified Data Engineer (for Spark, Delta Lake, ETL pipelines)
  - Mode Analytics / Advanced SQL courses (for database optimization)

## 3. Product & Project Management
* **Core Technical Skills**: Agile, Scrum, Kanban, Jira, Confluence, Product Roadmapping, User Research, Wireframing (Figma), SQL (basic), Google Analytics.
* **Core Soft Skills**: Stakeholder management, cross-functional leadership, negotiation, empathy, conflict resolution, prioritization.
* **Upskilling Pathways & Resources**:
  - Certified Scrum Product Owner (CSPO) / Certified ScrumMaster (CSM)
  - PMI Project Management Professional (PMP) (for corporate/traditional frameworks)
  - Product School Certification (for product strategy and metrics)
  - Pragmatic Institute Product Management Certification

## 4. Cybersecurity
* **Core Technical Skills**: Network security, penetration testing, vulnerability assessment, Linux, Wireshark, SIEM tools (Splunk), IAM, Cryptography, Firewalls.
* **Core Soft Skills**: Analytical thinking, incident response, crisis management, ethical judgment.
* **Upskilling Pathways & Resources**:
  - CompTIA Security+ (entry-level cybersecurity principles)
  - Certified Information Systems Security Professional (CISSP) (for senior/managerial cybersecurity roles)
  - Certified Ethical Hacker (CEH) (for penetration testing/red teaming)
  - Splunk Core Certified Power User (for SOC analysts)
